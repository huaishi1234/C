## test1:
    1、建立单向链表，表长任意；
    2、可交互输出单链表中的内容；
    3、编写算法计算出自己所建单链表的长度并输出；
    4、删除自己所建单链表中的第K个结点，并将剩余结点输出；
    5、将单链表倒排，输出结果。
